# Unsupervised Domain Adaptation with SAM-RefiSeR for Enhanced Brain Tumor Segmentation

This repository contains the research code for the paper  
**“Unsupervised Domain Adaptation with SAM-RefiSeR for Enhanced Brain Tumor Segmentation”**,  
accepted at **IEEE BIBM 2025**.

---

## Overview
- Task: Brain tumor segmentation (ET / TC / WT)
- Modality: MRI
- Framework: PyTorch
- Focus: Unsupervised domain adaptation and SAM-based pseudo-label refinement

---

## Notes
- This is research code and may require minor modifications to run.
- Dataset paths and configurations are experiment-specific.
- Pretrained weights and datasets are not included.
